# 寝てる間にThreadsで9投稿されてた自動投稿の仕組みを作った話

寝てる間にThreadsで9投稿されてた——自 動投稿の仕組みを作った話

朝、スマホを見たら自分のThreadsアカウントに9件の投稿が並んでた。

全部、寝てる間にAIが書いて投稿したもの。

「...これ、誰が書いたんだっけ？」

私のAIです。

朝起きてスマホを見たらThreadsに9件の投稿が並んでいるイラスト

毎日投稿、しんどくないですか？

SNSは「毎日投稿が大事」ってみんな言う。

sasuu｜非エンジニアのAI活用術 2026年3月15日 06:30

でも正直、ネタ出し → 書く → 投稿 → 反応確認のループがキツい。

特にThreads（Instagramの姉妹SNS。文字中心の投稿ができるやつ）は、1日に何件も投稿したほう が伸びやすい。手動で毎日9件？ 無理。

だから全部AIにやらせることにした。

仕組みはこう

ざっくり言うと、こんな流れ。

毎朝5時に、AIが9投稿分の文章を自動生成する

生成した投稿はデータベースに保存される

朝6時から夜10時まで、2時間おきに1件ずつ自動で投稿される

投稿フロー図：朝5時AI生成 → データベース保存 → 2時間おきに自動投稿

自分がやることは？ 何もない。寝てればいい。

...とまぁ、仕組みだけ聞くとスゴそうに聞こえるんですけど。

Threadsには「空気」がある

ここが面白いところで、Threadsって独特の文化があるんですよ。

X（旧Twitter）とは全然ノリが違う。

Xは「情報」「議論」「スピード感」。 Threadsは「共感」「優しい世界」「ゆるい日常」。

だからXで伸びる投稿をそのまま持っていっても、全然反応がない。

AIに投稿を作らせるにしても、Threadsの空気に合わせないとダメ。

具体的にやったこと。

AIに「キャラクター設計書」を渡した（一人称「僕」、語尾「〜だよ」「〜かも」）

「煽り」「論破」はNGと明記

投稿のパターンを3つに分けた（実況50%、問いかけ30%、ぽろり20%）

自分のX投稿をベースに、Threadsのトーンに変換する仕組みにした

で、結果どうだったか

正直に言います。

ちょっと気持ち悪かった。

いや、仕組みとしては動いてる。ちゃんと投稿される。文法も自然。

でも読み返すと...なんかこう...

「おじさんのつぶやき」みたいになってた。

「おじさんのつぶやき」的な投稿のイメージ（ちょっと気持ち悪い感じ）

具体的に何がマズかったか。

「今日も頑張ろうね」→ 誰に言ってるの？感

「ちょっと思ったんだけどさ」→ 距離感バグってる

「〜かもしれないなぁ」→ ぼんやりしすぎて何も伝わらない

要は「自分の声」がないんですよ。

AIが作った文章って、文法は完璧なんだけど「この人が言ってる感」がない。Threadsのユーザーって 共感で動くから、「中身のない丁寧な文章」は逆に浮く。

自動化できたこと、できなかったこと

ここまでやってわかったのは、自動化には「向いてる部分」と「向いてない部分」がハッキリあると いうこと。

自動化できた（助かった）こと

投稿の予約・実行 → 完全自動。二度と手を動かさなくていい

ネタ出しのたたき台 → AIが出してきたものを叩く方が、ゼロから考えるより100倍ラク

投稿時間の管理 → 2時間おきにスマホ触る必要がなくなった

自動化が微妙だったこと

文章のトーン → 「自分っぽさ」はAIだけじゃ出せない

感情の表現 → 「楽しい」「悔しい」の温度がAIだと一定すぎる

リアルタイム性 → 「今日こんなことがあった」は自動生成できない（当たり前だけど）

今のやり方

完全自動を諦めたわけじゃない。

今は「AIが下書きを9件作る → 自分が3分で見て微調整 → 予約投稿」の流れにしてる。

ゼロから9件書くのは30分以上かかる。 AIの下書きに手を入れるだけなら3分。

この差は大きい。

自分がやるのは「これ、自分が言いそうか？」のチェックだけ。 それ以外は全部AIに任せてる。

SNSで本当にやるべきことは「投稿を作ること」じゃなくて「返信すること」「つながること」だっ た。作る部分をAIに任せたおかげで、その時間が生まれた。

まとめ

自動投稿の仕組みは作れる。動く

でも「おじさんのつぶやき」にならないように、自分の声を入れる工程は必要

完全自動は微妙。「ほぼ自動+3分チェック」がちょうどいい

投稿の自動化で空いた時間で、返信やコミュニケーションに回すのが正解

...AIに全部やらせたら楽になると思ったのに、結局「自分っぽさ」は自分でしか出せないという、当 たり前の結論に着地しましたとさ。

でも、まだ諦めてない

ちょっと思ってることがあって。

Threadsって、昔のTwitterに近い空気があると感じてる。ビジネス感が薄くて、日常の延長で人と会話 する場所。

で、AIって「記事を書く」とか「情報を集めて整理して、わかりやすくまとめる」みたいな作業は人 間より圧倒的に強い。私が毎日noteを書けてるのもAIのおかげ。

でも「会話を生む文章」——誰かが思わず返信したくなるような一言を作るのは、正直まだ自分の力 量では難しいなと。

ただ、面白い事例がある。

AIの活用で有名な深津さん（THE GUILDの代表）がXで「スーパーエコちゃん」っていう自動投稿ア カウントを運用してるんですよ。完全にAIが動かしてるんだけど、これがちゃんとエモい。感情のあ る言葉が出てくる。

「あ、ロボットがやってるんだけど、なんか人間っぽいな」って思わせる温度感がある。

私のThreads自動投稿が「おじさんのつぶやき」になっちゃったのは、キャラクター設計の作り込みが まだ甘かったんだと思う。

次のステップとして、スーパーエコちゃんみたいな事例を参考にしながら、Threadsのキャラクターを もっと深く設計していきたい。「自分っぽさ」をAIに再現させるんじゃなくて、「AIが演じるキャラク ター」として独立させる方向。

この続きは、また進展があったら書きます。
