ターミナルからGoogleを丸ごと操作するgogcli

ひとことで言うと、 ターミナル（黒い画面）からGoogleを丸ごと操作できるツールです。

ブラウザを開かなくても、文字を打つだけで メール送信もカレンダー登録もできちゃいます。

2026年1月にv0.9.0がリリースされたばかりで、 今めちゃくちゃ話題になってるんですよね。

OpenClawエコシステムでも紹介されていて、 開発者コミュニティでも注目度が高いです。

AIとの組み合わせがすごい

会話するだけでGoogle操作完了：AI×gogcliの連携フロー

ここからが本題です。

gogcliの真価は、AIと組み合わせたときに発揮されます。 Claude Codeと連携させると、 「AIが Googleを操作してくれる」 状態になります。

たとえばこんな感じ。

例1: カレンダー確認 自分：「今日の予定を教えて」 → AIがgogcliでカレンダーを読み取って、そのま ま回答してくれる

例2: スプレッドシート操作 自分：「このスプレッドシートに今月の売上を追記して」 → AIがgogcliで Sheetsに書き込んでくれる

ブラウザを開く必要すらありません。 会話するだけで、Googleの操作が完了します。

ほかにも使い道はたくさんあって、

「未読メールの中で急ぎのものだけ教えて」

「来週の空き時間をまとめて」

「ドライブにある請求書を探して」

みたいなことも、自然な日本語で頼むだけ。 AIが裏側でgogcliを叩いて、結果を返してくれます。

gog drive list --query "name contains 'レポート'"

従来の方法との違い

これまでAIにGoogleを操作させるには、 サービスごとに別々の接続設定（MCPサーバー）が必要でし た。

これがまぁ面倒なんですよね。 設定ファイルを何個も書いて、トークンを取得して、 エラーが出たら どこが原因か特定するのも一苦労。

gogcliなら、1つのツールで全部OK。 最初に1回だけGoogleアカウントでログインすれば、 あとは全 サービス使い放題です。

シンプルだから、トラブルも少ない。 まぁこれが一番ありがたいかなと。

実際にやってみた

試しにカレンダーの予定を取得してみました。

こんな感じで、予定がバッと一覧で返ってきます。 Claude Codeからこれを呼び出せば、 「今日の予 定を教えて」と聞くだけで答えてくれるわけです。

便利すぎませんか。

導入はびっくりするほど簡単

インストール1行コピペ、認証1回ログイン、以上これだけ

「すごいのはわかったけど、設定が大変なんでしょ？」

いえ、びっくりするほど簡単です。

ステップ1: インストール ターミナルに1行コピペするだけ。

$ gog calendar events

ID START END SUMMARY ... 2026-02-10T22:30:00+09:00 2026-02-11T05:30:00+09:00 寝る ... 2026-02-11T10:00:00+09:00 2026-02-11T10:30:00+09:00 🏋️ エニタイムフィットネス解約手 ... 2026-02-11T19:00:00+09:00 2026-02-11T20:00:00+09:00 AI部チームTWO ... 2026-02-11T20:30:00+09:00 2026-02-11T21:00:00+09:00 AIエージェント定例

これだけです。

ステップ2: 認証 Googleアカウントで1回ログインするだけ。 ブラウザが開くので、許可ボタンを押せ ば完了します。

以上。本当にこれだけ。

Macをお使いの方なら、Homebrewが入っていればすぐ始められます。 Windowsの方はWSL経由で利 用できるので、そちらもチェックしてみてください。

詳しいセットアップ手順や、 Claude Codeとの連携方法は公式サイト（gogcli.sh）に載っています。 わからない箇所はChatGPTなどで補完しながら進めてみてください。
