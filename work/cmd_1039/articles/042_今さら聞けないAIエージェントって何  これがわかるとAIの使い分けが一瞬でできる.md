「AIエージェント」ってなに？ — 掃除ロボットで考え

てみて

ChatGPT=物知りな友達（教えてくれるけど掃除はしない） vs AIエージェント=ルンバ（勝手にやってくれる）の比較図

いきなり横文字出してすみません。 「エージェント」って言われても「なに？」ってなりますよね。

でも、身近なもので考えると一発で分かります。

ChatGPT = 物知りな友達。

「この汚れ、どうやって落とす？」って聞いたら教えてくれる。 でも、掃除はしてくれない。 自分で やるしかない。

AIエージェント = ルンバ（掃除ロボット）。

「掃除して」って言ったら、自分で動き出す。 部屋を見回して、どこが汚いか判断して、実際に掃除 する。 終わるまでずっと動き続ける。

この違い、伝わりますかね。

ポイントは、AIエージェントが自分でこのループを回すこと。

見る → 考える → 動く → また見る → ...

人間は「やって」と言うだけ。 途中で「ここどうしますか？」なんて聞いてこない。 自分で判断し て、最後まで走り切る。

たとえば、こんなことができます。

領収書のファイルを読んで → 内容を整理して → Excelにまとめる

フォルダの中身を見て → 内容を理解して → 資料を自動で作る

「このWebサイトのデザイン作って」→ 画面レイアウトを考えて → コードを書いて → 完成

全部、AIが勝手にやってくれる。 人間がやったのは「お願い」だけです。

ちなみに動画でバズってるやつとか見ると、もっとすごいです。 領収書100枚を一気に処理して経費 精算完了、みたいな。 人間がやったら半日かかる作業が、数分で終わる。

「脳」と「手足」— AIエージェントの仕組み（超ざっく

り）

AIエージェントの仕組み図（脳=考える力 + 手足=動く力 → スキル=仕事のやり方マニュアル。1回教えれば何度でもやってくれる）

もうちょっとだけ中身の話をさせてください。 難しくないです。2つだけ覚えればOK。

AIエージェントには「脳」と「手足」があります。

脳 = 考える力。

「丁寧語で答えてね」「日本語で報告してね」みたいな指示書を渡す。 すると、それに従って考えてく れます。

人間が「こうやって仕事してね」って伝える部分。 会社で言えば、業務マニュアルみたいなもので す。

手足 = 実際に動く力。

パソコンの中のファイルを読んだり書いたりする力

外のアプリ（Googleカレンダーとか）と連携する力

脳だけだと「考えるだけ」。 手足だけだと「何も考えず動くだけ」。

両方あって、はじめて「仕事ができる」んです。

で、この脳と手足を組み合わせたものが「スキル」。

ざっくり言うと、仕事のやり方マニュアルです。

新人バイトにマニュアルを渡すのと同じ感覚。 最初に1回教えれば、あとは同じ品質で何度でもやっ てくれる。

しかも文句言わないし、24時間働ける。 深夜3時でも嫌な顔しない。

...まぁ、たまにミスはしますけどね（笑） でも「ここ違うよ」って言えばすぐ直してくれます。

人間の部下より素直かもしれない。

自分はこうなった — 「聞く」から「任せる」へ

夜「これ作って」と入力して寝る→朝起きたら完成品ができている図
