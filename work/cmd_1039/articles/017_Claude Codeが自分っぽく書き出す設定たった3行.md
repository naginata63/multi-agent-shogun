Claude Codeが"自分っぽく"書き出す設定、 たった3行

Claude Codeに文章を書いてもらったとき、「なんか自分っぽくないな」って思ったことありません か？

私もそうでした。出てくる文章がどれも丁寧すぎて、自分の書き味とは程遠い。「させていただきま す」が並ぶ文章を見るたびに、これ誰が書いたの？って感じでした。

でも、Claude.mdというファイルに　3行書くと"自分っぽい"文章をかけるようになります。。

なんでそんなに堅いの？

【画像: AIが書く文章が堅すぎる問題 (01-stiff-response.png)】

Claude Code（パソコンの画面で動くAIアシスタント）に記事の下書きを頼むと、初期状態だとこんな 文章が出てきます。

「〜させていただきます」 「〜でございますが」 「ご確認いただけますと幸いです」

...自分のnote記事にこんな文体、使わないですよね。

sasuu｜非エンジニアのAI活用術 2026年3月11日 06:30

「もうちょっと自分の口調に寄せてほしいんだけどな」と思って、毎回「もっとカジュアルに書いて」 と指示。

カジュアルって言っても漠然としてるし、毎回言うのも正直めんどう。

実は、設定ファイルに書いておけば一発で変わります。

CLAUDE.mdに3行書くだけ

【画像: CLAUDE.mdに口調ルールを書く手順 (02-claude-md-setup.png)】

Claude Codeには 「CLAUDE.md」 という設定ファイルがあります。

ざっくり言うと「AIへの指示書」です。 ここに書いた内容を、AIが毎回読んでから文章を書いてくれ ます。

ということで、やることはシンプル。

まず、自分のプロジェクトフォルダに CLAUDE.md というファイルがあるか確認してください。

ない人 → Claude Codeを開いて

/init

と打つ。これだけでCLAUDE.mdが自動で作られます。

もうある人 → Claude Codeに「claude.mdってなんて書いてある？」って聞いてCLAUDE.mdを読み込 ませつつ自分でもどんな風に書かれているか確認してみてください。

後は、Claude Codeに追加でこう頼むだけ。

その下に、自分のnote記事やブログから「これが自分っぽいな」と思う文章を3つくらいペタッと貼 る。 貼り付けることができない場合は、ちょっと面倒ですがファイル名を入力して「これらのファイ ルを確認して」と付け足すだけでOK。

すると、Claude Codeがあなたの書き味を分析して、こんな感じのルールを自動で作ってくれます。

自分が過去に書いた文章を3つ貼るので、この書き味を分析して、CLAUDE.mdに書く「書き味ルール」を作ってく

CLAUDE.mdに上記のような新しいルールが追加されていたら、以降は書いてもらう文章の口調や文体 が変わっています。

ビフォーアフター

【画像: 設定前と設定後の書き味の変化 (03-before-after.png)】

設定前:（AIに「この機能の紹介文を書いて」と頼んだ場合）

本機能をご活用いただくことで、作業効率の大幅な向上が期待されます。ぜひご検討いた

だけますと幸いです。

設定後:

この機能、使ってみるとマジで作業が速くなります。まずは試してみてください！

たった3行で、自分の書き味にぐっと近づきました。

まとめ

【画像: 自分だけのAIアシスタントに育てる (04-summary.png)】

CLAUDE.mdに書き味ルールを追加するだけ。それだけで、Claude Codeが書く文章が"自分っぽく"な ります。

15分もかかりません。ぜひ試してみてください！

## 書き味ルール（例: Claudeが自動生成したもの） - 一人称は「僕」で、カジュアル敬語（「〜ですね」「〜しましょう」） - 「させていただきます」「でございます」は記事で使わない - 迷ったら友達に話すくらいの距離感で

ちなみに、書き味以外にも「記事の構成はこの順番で」とか「専門用語は噛み砕いて」みたいな指示 も書けます。

自分だけのライティングパートナーに育てていく感覚、なかなか楽しいですよ。
