Claude Code 超初心者講座 #8｜CSVやデ ータの集計・変換 — Excelを開かずに処理 する

前回は「直す」をやりました。 今回は「集計する」です。

CSVを開いて、列を選んで、SUM関数を組んで、フィルターかけて...。 心当たり、ありませんか。

Claude Codeなら、「売上の合計出して」で終わります。 Excelを起動する必要すらないです。

CSVの集計 — 「売上の合計出して」

CSV集計：CSVファイルから月別売上合計を一瞬で算出

sasuu｜非エンジニアのAI活用術 2026年2月17日 15:00

まずは一番よくあるやつ。 CSVファイルの数字を集計する場面です。

たとえば、売上データのCSVがあったとして。

「このCSVの売上合計を月別に出して」

これだけで、結果が返ってきます。

ExcelでいうSUM、AVERAGE、COUNTに相当する処理が、日本語だけで完結する。 関数を覚える必 要も、セル範囲を指定する必要もありません。

しかも条件付きの集計もいけます。

「東京支店だけ集計して」

「4月以降のデータだけで平均出して」

「カテゴリ別に件数を数えて」

ピボットテーブルを組んでいた作業が、一言で片付く。 正直、数字まわりはこれが一番ありがたいか なと。

データ変換 — 「JSONをCSVにして」

データ変換：JSON形式のデータをCSVに一発変換

次は地味だけど、手動でやると泣きたくなるやつ。 データの形式変換です。

月別売上: 1月: 1,250,000円 2月: 1,480,000円 3月: 1,120,000円 合計: 3,850,000円

たとえば、WebサービスからダウンロードしたデータがJSON形式だったとします。 これ、Excelで開 けないんですよね。

「このJSONデータをExcelで開ける形にして」

こう頼むだけで、CSVに変換してくれます。

対応している変換パターンはこんな感じ。

JSON → CSV

CSV → JSON

TSV → CSV

その他、テキストベースならだいたいOK

エンコーディングの変換もやってくれます。 「文字化けするんだけど」と伝えれば、UTF-8からShift- JISへの変換も勝手にやってくれる。

Webからコピペしたデータがぐちゃっとしてるとき、ありますよね。 そういうのを「CSVにまとめ て」と投げるだけで、整った表形式になって返ってきます。

データ整形 — 「グラフ用に整えて」

データ整形：ぐちゃぐちゃなデータをキレイな表に自動整形

最後は「見せる」ための準備。 集計したデータをプレゼンやグラフに使いたいときの話です。

「このデータをピボットテーブルみたいにして」

こう頼むと、行と列を入れ替えたり、クロス集計してくれたりします。

他にもこんな指示が通ります。

「不要な列を削除して、売上と日付だけ残して」

「列名を英語から日本語にして」

「金額が大きい順に並べ替えて」

「プレゼン用にキレイにまとめて」

特に「プレゼン用に」が便利で、集計結果を見やすい表形式に整えてくれる。 そのまま資料に貼れる レベルの出力が返ってきます。

整形が終わったら「CSVで保存して」と伝えれば、ファイルとして書き出してくれる。 あとはExcel やGoogleスプレッドシートで開くだけです。
