インストールに必要なもの一覧（Mac・ターミナル・Node.js・APIキー）
始める前に、これだけ用意してください。
Mac（WindowsやLinuxでもOK。今回はMacで説明）
ターミナル（パソコンに文字で命令を送る画面。Macなら「ターミナル」アプリを開くだけ。最初
から入ってます）
Node.js 22以上（プログラムを動かすための土台ソフト。これがないとOpenClawが動きません。
入ってなければ公式サイトからインストール）
AIのAPIキー（AIサービスを使うためのパスワードのようなもの。Claude、GPT、GLMなど）
APIキー（AIサービスのパスワード）は、使いたいAIサービスの公式サイトで取得できます。 まだ持っ
てない人は「Claude APIキー 取得方法」とかでググればすぐ出てくる。

"gateway": {"mode": "local"} が入ってないと、LINEなど外部と連携するための「受付窓口（Gateway）」
が起動できません。 もし入ってなかったら手動で追記してください。
ここ、最初のハマりポイントです。自分も最初これでつまずいた。
ステップ4: ヘルスチェック
こちらもコピーして貼り付けるだけです。
これがめちゃくちゃ便利。 足りない設定や壊れてる部分を自動で検出して、修正までやってくれま

初回は必ず実行してください。
ステップ5: 受付窓口を起動する
こちらもコピーして貼り付けるだけです。
これはOpenClawの「受付窓口」を起動するコマンドです。 LINEやDiscord、チャット画面など、いろ
んな入り口からのメッセージを受け取って、AIに橋渡ししてくれます。
別のターミナルウィンドウで起動してください。（Macなら「ターミナル」アプリをもう1つ開くだ
け） これを裏で常に動かしておくのが基本です。
ステップ6: 対話モードを起動
こちらもコピーして貼り付けるだけです。
openclaw doctor
openclaw gateway

<!-- page 5 -->

文字だけのチャット画面が立ち上がります。LINEの文字版みたいなものです。 ここにメッセージを打
ち込めば、AIが返事してくれる。
「今日の天気は？」とか打ってみてください。 返事が返ってきたら、セットアップ完了です。
よくあるハマりポイント
よくあるハマりポイント3つ（①openclawだけ打ってもヘルプのみ → TUIが正解、②Gateway未起動 → 先にgateway、③Gateway再起動のコ
マンド）
セットアップで詰まりやすいポイントをまとめておきます。
1. openclaw だけ打ってもヘルプが出るだけ
対話モードは openclaw tui。 openclaw 単体はコマンド一覧を表示するだけです。
2. チャット画面を起動する前に受付窓口が必要
受付窓口が動いてないと、チャット画面がメッセージを送れません。 先に openclaw gateway を別の
ターミナルウィンドウで起動してから。
3. 受付窓口の再起動の方法
以下をそのままコピーして貼り付けるだけでOKです。
openclaw tui
openclaw gateway stop && openclaw gateway
