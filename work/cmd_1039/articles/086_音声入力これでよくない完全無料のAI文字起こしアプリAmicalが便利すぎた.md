Amicalって何？

ざっくり言うと、AIが賢くしてくれる音声入力アプリ。

裏側ではOpenAIのWhisper（音声認識）とLLM（文章整形）が動いていて、 喋った内容をいい感じに テキスト化してくれる。

で、これが完全無料。MITライセンスのオープンソース。 サブスク課金とかない。マジで。

有料の音声入力アプリって月1,000〜2,000円くらい取るものもあるけど、 Amicalはそれと同等かそれ 以上のことが、タダでできる。

何がいいのか

1. アプリに合わせて文体が変わる（らしい）

公式によると、Gmailならビジネスっぽく、Discordならカジュアルに、 みたいにアプリを検知してト ーンを変えてくれるらしい。

とはいえ自分は、自分の話し方でそのまま書き起こしてもらうスタイルなので、 この機能はあまり使 っていない。

でも「メールの時だけ丁寧語にしたい」みたいな人には便利かも。

2. ローカルでもクラウドでも選べる

Amicalは2つのモードがある。

ローカルモード: Whisper + Ollamaで完全オフライン動作。 音声データが外部に一切出ない。プライ バシー重視の人向け。

クラウドモード: OpenAIやOpenRouterのAPIを使う。 正直、こっちの方が速いし精度も高い。

個人的にはクラウドモードを推したい。 ローカルだとどうしてもマシンスペックに左右されるし、 ク ラウドの方がレスポンスが明らかに速い。精度も安定してる。

「プライバシーが気になる」って人はローカルでもいいと思うけど、 速度と精度を求めるなら素直に クラウド一択かなと。

で、ここが大事なんだけど、 どっちのモードでもAmical自体は無料。 クラウドモードの場合はAPIの 利用料だけかかるけど、 アプリに課金は一切なし。

3. 50言語以上対応

日本語はもちろん、英語、中国語、韓国語... 多言語ミックスでも認識してくれる。

日本語の中に英単語が混ざる、みたいなのも普通に拾える。

4. カスタム辞書が作れる（けど、そこまで必要ない）

一応、カスタム辞書機能がある。 よく使う単語を登録しておけるやつ。

ただ正直、そこまで精度が高くない。 「この発音をしたらこの漢字を出す」みたいなきめ細かい設定 ができるわけじゃないので、 期待しすぎるとちょっと肩透かしかも。

とはいえ、そもそもAI自体がかなり賢くなっているので、 多少の誤変換は文脈でちゃんと直してくれ る。

なので、カスタム辞書はあまり気にしなくていいと思ってる。 よっぽどひどい変換が繰り返されるな ら登録すればいい、くらいの温度感。

実際に使ってみて — AquaVoiceからの乗り換え

もともとAquaVoiceっていう音声入力アプリを使ってた。

AquaVoice、精度は結構良かったんだけど、 月額1,600円くらいかかるんですよね。

で、Amicalを試してみたら、 正直そこまで精度に差を感じなかった。

「え、これタダでできるの？」ってなった。 月1,600円浮くのは普通にデカい。

brew install --cask amical

Mac標準 Amical 精度 そこそこ Whisper + AI補正で高い 文脈理解 なし アプリごとに文体変更 プライバシー Appleに送信 ローカル or クラウド選択可 カスタマイズ ほぼなし 辞書・コマンド自由 価格 無料 無料（アプリ代0円）

一つだけ違いがあるとすれば、 AquaVoiceは喋りながらリアルタイムで文字が出てくる。 Amicalはそ こがちょっと違って、喋り終わってからまとめて変換される感じ。

ただ、個人的にはリアルタイムで文字が出てほしい って場面があまりなかった。

喋る → テキストになる → 使う。

このフローで十分だったので、Amicalで全然問題なかった。

月1,600円払ってたのと同じことが、タダでできてる。

これだけで乗り換える理由としては十分かなと。
