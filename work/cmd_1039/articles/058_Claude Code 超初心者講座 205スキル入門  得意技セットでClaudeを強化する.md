スキルとは — Claudeに得意技を追加するイメージ
普段のClaude Codeは「何でも屋」です。 質問すれば答えてくれるし、ファイル操作もしてくれる。
でも「デザインを整えて」とか「テストを書いて」みたいな特定の作業になると、毎回同じような指
示を出すことになりますよね。
スキルは、そういうよく使う指示のセットをひとまとめにしたもの。 一度追加すれば、短いコマンド
一つで呼び出せます。
たとえば：
こう打つだけで、Claudeがデザインの定石を踏まえた上で画面を整えてくれる。 色のバランス、余白
の取り方、ボタンの動きまで。 自分で一つずつ指示しなくても、いい感じに仕上がります。
どこから追加するの？ — スキルストア
スキルストア — アプリストアのようにスキルを探して追加
/frontend-design ToDoアプリをモダンな感じにして

<!-- page 3 -->

スキルはClaude Codeのスキルストアから追加できます。 スマホでアプリストアからアプリを入れる
のと同じ感覚です。
やり方はシンプル。Claude Codeを開いて、こう打つだけ。
これでスキルストアが開きます。 一覧から好きなスキルを選んで追加するだけ。
たとえばこんなスキルが公開されています：
frontend-design — 見た目をプロっぽく整える
pdf — PDFファイルの読み取り・編集
pptx — プレゼン資料の作成
docx — Word文書の作成・編集
webapp-testing — Webアプリの動作確認
追加したらすぐ使えます。 必要なものだけ入れればOK。アプリと同じで、入れすぎると重くなるので
注意。

「毎回同じことを頼んでるな」と感じたら、それをスキルにするチャンス。
作り方もClaude Codeに頼むだけ。
Claudeが
.claude/commands/
というフォルダにスキルを作ってくれます。 次からは
/daily-report
のように短いコマンドで呼び出せる。
よく使う作業を自分専用のコマンドにできる。 これがスキルの一番の強みです。
