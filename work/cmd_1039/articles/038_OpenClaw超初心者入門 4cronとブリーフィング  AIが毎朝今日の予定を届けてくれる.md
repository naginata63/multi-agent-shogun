cronの概念図（目覚まし時計のアイコン → 毎朝7時に予定まとめ・毎日18時にX通知・毎週月曜にタスク整理の3つが時計から放射状に配置）
cron（クロン）は、目覚まし時計のように、決めた時間に自動で仕事をさせる仕組みです。
「毎朝7時になったら、この作業をやっておいて」とAIにお願いしておける、タイマー予約のようなも
のです。
毎朝7時に「今日の予定をまとめて」
毎日18時に「Xの通知を要約して」
毎週月曜に「今週のタスクを整理して」
こういう定期的な仕事を、OpenClawが自動でやってくれます。

ステップ3: 翌朝を待つ
...いや、待てないですよね。わかる。
テストしたい場合は、以下をコピーして貼り付ければ今すぐ手動で実行できます。
LINEに結果が届いたら、設定完了です。おめでとうございます!
openclaw cron add \
  --name "Morning brief" \
  --cron "0 7 * * *" \
  --tz "Asia/Tokyo" \
  --session isolated \
  --message "今日の予定、天気、メールの未読、GitHubの通知をまとめて" \
  --channel line
openclaw cron list
openclaw cron run "Morning brief"

cronの時間指定の書き方
cron式の読み方（「分・時・日・月・曜日」の5つの枠に数字を入れる図。例: 0 7 * * * = 毎日7:00）
"0 7 * * *" の部分、暗号みたいに見えるかもしれません。 でも仕組みはシンプルです。5つの数字で
「いつ実行するか」を指定しているだけです。
*（アスタリスク）は「毎回」「いつでも」という意味。 左から「分・時・日・月・曜日」の順番で
す。
最初は「毎日〇時」のパターンだけ覚えておけば大丈夫です。上の例をコピーして数字を変えるだけ
でOK。
使える定期タスクの例
定期タスクの例を3カテゴリで整理（情報収集系: 天気・メール・X通知、メンテナンス系: デスクトップ整理・ダウンロード掃除、通知系: 為替
レート・Web更新監視）
ブリーフィング以外にも、いろいろ使えます。
情報収集系
毎朝: 天気予報 + カレンダー + メール要約
分  時  日  月  曜日
0   7   *   *   *     = 毎日7:00
0   18  *   *   *     = 毎日18:00
0   9   *   *   1     = 毎週月曜9:00
*/30 *  *   *   *     = 30分ごと

毎夕: X（Twitter）の通知まとめ
毎週月曜: 仕事のタスクリストの棚卸し
メンテナンス系
毎日: デスクトップの不要ファイル整理
毎週: ダウンロードフォルダのクリーンアップ
通知系
毎朝: 為替レートをチェックして報告
定期的: 特定のWebサイトの更新を監視
自分の生活に合わせて、好きなだけ追加できます。
ChatGPTとの違い
ChatGPTとOpenClawの違い（左: ChatGPT「自分から聞きに行く」矢印が人→AI、右: OpenClaw「AIから届く」矢印がAI→人。受け身でOK
の図）
ChatGPTには定期実行の仕組みがありません。
毎朝自分で開いて「今日の予定まとめて」と打たないといけない。 つまり、自分からアクションを起
こす必要がある。
OpenClawは逆。 AIの方から勝手に情報を届けてくれる。
この「受け身でOK」なのが、地味にデカい。
朝起きてLINEを開いたら、もう今日のブリーフィングが届いてる。 何もしなくていい。
