LINE連携の仕組み（スマホ → LINE → Messaging API → OpenClaw Gateway → AI → 返事がLINEに戻る流れ）
ざっくりこういう流れです。
LINEでメッセージを送ると、LINEの公式機能を通じて、OpenClawの受付窓口に届きます。そこから
AIが内容を処理して、返事がまたLINEに戻ってくる。
外出先でも、通勤中でも、布団の中からでも。 LINEさえ開ければ、自宅のPCにいるAI秘書に話しか
けられます。
設定手順
LINE連携の設定4ステップ（①LINE公式アカウント作成 → ②openclaw.jsonにトークン追記 → ③Webhook URL設定 → ④Gateway再起動 → テ
スト送信）
ステップ1: LINE公式アカウントを作る
LINE Developers（https://developers.line.biz）というLINEの開発者向けサイトにアクセスして、 ボッ
ト用のアカウントを作ります。
[あなたのスマホ] → LINE → [LINE Messaging API] → [OpenClaw Gateway] → [AI]

<!-- page 3 -->

1. LINE Developersにログイン（普段のLINEアカウントでOK）
2. 新しいプロバイダー（開発者名のようなもの）を作成
3. 「Messaging API」チャネルを作成
4. チャネルアクセストークン（LINEとOpenClawを繋ぐための合言葉）を発行
このトークンが、OpenClawとLINEをつなぐ鍵になります。
ステップ2: OpenClawにLINEチャネルを追加
OpenClawの設定ファイル（~/.openclaw/openclaw.json）に、さきほど取得したトークンを書き加えま
す。
以下の内容をそのままコピーして貼り付け、"ここに〜" の部分だけ自分のトークンに置き換えればOK
です。
ステップ3: メッセージの届け先を設定する
LINEからのメッセージを自動で受け取るための「届け先」を設定します。 郵便受けのようなもので、
「メッセージが届いたら、ここに転送してね」とLINEにお願いする設定です。
LINE Developersの画面で、転送先のURLを設定します。
自宅のPCで動かす場合は、ngrok（エングロック）というツールを使います。 ngrokは、自分のパソコ
ンをインターネットに一時的に公開するツールです。普通、自宅のPCにはインターネット上の「住
所」がないのですが、ngrokが仮の住所を発行してくれます。
{
  "channels": {
    "line": {
      "token": "ここにチャネルアクセストークン",
      "secret": "ここにチャネルシークレット"
    }
  }
}
https://あなたのドメイン/webhook/line

実行すると画面にURLが表示されるので、それをLINE Developersの「Webhook URL」の欄にコピペ
してください。

ステップ5: テスト
LINEで自分のBotアカウントにメッセージを送ってみてください。
「今日の天気は？」
返事が返ってきたら、連携完了です。おめでとうございます!
知っておくべき制限
LINEフリープランの制限と節約術（月200通制限・グループ活用で節約・有料プラン月5,000円で15,000通）
LINEのボット機能（Messaging API）のフリープランには、月200通のメッセージ送信制限がありま
す。
ngrok http 18789
openclaw gateway stop && openclaw gateway

<!-- page 5 -->

「え、200通？すぐ使い切るじゃん」
って思いますよね。節約方法があります。
LINEグループを活用する。
グループ内でのメッセージはカウントの仕方が違うので、 1対1よりも効率的に使えます。
本格的に使い倒すなら有料プラン（月5,000円で15,000通）も視野に入ってくる。 でもまずはフリープ
ランで十分試せます。
LINEで何ができる？
LINEからできること一覧（ファイル送信・カレンダー確認・GitHub確認・Web検索＆要約の4つのアイコン付き）
連携が終わると、こんなことができるようになります。
「あのファイルをメールで送って」
「今日のカレンダー教えて」
「仕事のタスク管理の状況まとめて」
「〇〇のWebサイトを調べて要約して」
つまり、PCの前にいなくてもAIに仕事を頼める。
外出中に「あ、あれやっておきたい」と思ったら、 LINEで一言送るだけ。
帰宅したらもう終わってる。
