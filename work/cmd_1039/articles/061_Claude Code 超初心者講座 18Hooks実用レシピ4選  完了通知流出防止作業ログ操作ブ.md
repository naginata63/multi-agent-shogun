うっかり流出防止 — 保存時にパスワードやAPIキーを検知してブロック
AIにコードを書いてもらっていると、たまにパスワードやAPIキーがそのままファイルに残ってしまう
ことがあります。
そのまま公開したら大事故。
こう頼んでおけば防げます。
これで、うっかり大事な情報を含んだまま保存しようとすると自動でストップがかかります。
自分では気づかないミスをAIが防いでくれる。 特にコードを扱う人は、とりあえず設定しておいて損
はないです。
