そもそも「AIエージェントの住所」って何？

OpenClaw（自分の代わりに動いてくれるAIアシスタント）を使うとき、「どこで会話するか」を決め る必要があります。

メールを送る、ファイルを整理する、Webで調べ物をする——そういう作業をAIに頼むわけですが、 その「頼む窓口」がメッセージアプリです。

LINEで頼んでもいいし、Discordで頼んでもいい。

ただ、アプリによって「AIとの相性」がぜんぜん違う。

ここが今回の話のキモです。

候補は6つ

比べたのはこの6つ。

LINE — 日本で9,500万人が使ってる王者

Discord — ゲーマーやコミュニティの定番。基本無料

Slack — ビジネス向け。会社で使ってる人も多い

Chatwork — 日本のビジネス向けチャット

Microsoft Teams — 大企業御用達

Telegram — 日本では超マイナー。でも世界では10億人が使ってる

結論から言います

AIエージェントの「個人アシスタント」用途なら、Telegramが最適解でした。

「えっ、なんで？」ってなりますよね。

理由は3つあって、全部「お金」と「手間」と「制限」の話です。

理由1：完全無料。メッセージ何通送っても0円

【画像: Telegram無料無制限 vs LINE月200通の蛇口メーター比較 (01-free-vs-paid.png)】

ここが一番デカい。

Telegramの「ボットを動かすための仕組み」は、2015年のリリースからずっと完全無料です。

1日100通送ろうが、1,000通送ろうが、お金はかからない。

一方、LINEはどうか。

LINEの公式アカウント（ボットを動かすのに必要）には「メッセージ上限」があります。

月200通。

AIエージェントって、ちょっとした確認や報告も含めると1日20〜30通くらい平気で送ってきます。

1週間持たない。

「じゃあ月5,000円払えば？」って話ですが、AIに話しかけるためだけに月5,000円。 しかもAI本体の 利用料（Claude ProやChatGPT Plusの月3,000円）とは別で。

……高くない？

Telegramなら、0円。

理由2：自宅のパソコンだけで動く。サーバー不要

【画像: 呼び出し電話型vs自分で見に行く型の対比図 (02-polling-vs-webhook.png)】

LINE料金プラン（2026年現在） ━━━━━━━━━━━━━━━━━━━━━ 無料プラン: 月200通まで（1日たった6〜7通） ライト: 月5,000円で5,000通 スタンダード: 月15,000円で30,000通 ━━━━━━━━━━━━━━━━━━━━━

ここ、技術的だけど大事なポイントです。

メッセージアプリとAIをつなげるとき、方式が2つあります。

方式A：「呼び出し電話」型（Webhook） → メッセージが来たら、アプリ側からあなたのパソコンに 電話してくる → 電話を受けるには、ネット上の「住所」と「身分証明書」にあたるものを用意する必 要がある → つまりサーバーを借りるか、特殊な設定が必要

方式B：「自分で見に行く」型（Long Polling） → あなたのパソコンが定期的に「新しいメッセージ ある？」と聞きに行く → 自宅のWi-Fiでそのまま動く → サーバー不要。特殊な設定も不要

Telegramは方式B（Long Polling）がデフォルト。

LINEは方式A（Webhook）が必須。

つまりLINEでAIエージェントを動かすなら、自分のパソコンだけじゃ足りない。 外部にサーバーを借 りるか、「トンネル」っていう迂回ルートを作る必要がある。

Telegramなら、自宅のMac miniやノートPCだけでそのまま動く。

これ、「自宅でAIを24時間動かしたい」って人にとっては決定的な差です。

理由3：セットアップが異常に簡単

【画像: セットアップ時間の横棒グラフ比較 (03-setup-time.png)】

Telegramのボット作成手順：

Telegramで「@BotFather」に話しかける

「/newbot」と送る

名前を決める

トークン（パスワードみたいなもの）がもらえる

OpenClawの設定ファイルにそのトークンを貼り付ける

起動。終わり

所要時間：5分くらい。

審査なし。申請なし。お金なし。

LINEだと「公式アカウント作成 → 開発者登録 → 接続設定 → セキュリティ証明書の準備」で30分〜1 時間。

Slackだと「管理画面でアプリ作成 → ログイン許可の設定 → アクセス範囲の設定」で15〜30分。

Teamsだと「マイクロソフトのクラウド登録 → ボット設定 → アプリ定義ファイル作成」で……数時 間コースです。

じゃあ他のアプリはダメなの？

ダメじゃないです。用途によっては他が最適になります。

Discord — チームで使うなら最強

Discordのボット作成も無料だし、メッセージ制限もない。

「じゃあTelegramじゃなくてDiscordでよくない？」

うん、チームやコミュニティで使うならDiscordが最適です。

Discordの強みは「サーバー → チャンネル → スレッド」っていう階層構造。

「#開発用」「#質問用」「#雑談」みたいにチャンネルを分けて、それぞれにAIエージェントを配置でき る。

ただ、個人アシスタントとして使うには構造が大げさ。

「今日のスケジュール教えて」って聞くためにサーバーを立てるのは、1人暮らしなのに会議室を借り るようなもの。

Telegramなら、友達にLINEするような感覚でAIに話しかけられます。

Slack — 会社で使ってるならアリ

仕事のやりとりが全部Slackに集まってて、「その延長線上にAIアシスタントがいてほしい」なら、 Slackは良い選択です。

ただし注意点が2つ。

1つ目：無料プランだと90日でメッセージが消える

Slackの無料プランは、90日以上前のメッセージが検索できなくなります。

AIエージェントの記憶が90日で消える。ちょっと困る。

有料プラン（月$7.25/ユーザー〜、約1,100円）なら無制限ですが、個人で使うにはやっぱりコストが 気になる。

2つ目：無料プランはアプリ10個まで

AIエージェント + カレンダー連携 + 通知ボット + ……とやっていくと、あっという間に10個の枠を使 い切ります。

Chatwork — AIエージェントには向かない

レート制限（1分間に送れるメッセージ数の上限）が5分で300回。

1秒に1回。

AIエージェントが「調査結果をまとめて送るね」って10個のメッセージを一気に送ろうとすると、す ぐ制限に引っかかる。

しかもOpenClawは公式には非対応。無理やりつなげることはできるけど、茨の道です。

Microsoft Teams — 個人利用は論外

企業のIT部門が管理するツールです。

ボット作成にAzure（マイクロソフトのクラウドサービス）の登録が必要。 しかも開発ツールの世代 交代中で、公式のやり方が安定していない状況です。

会社で「Teamsに社内AIボットを入れたい」なら検討の余地ありますが、個人のAIアシスタント用途 では選ぶ理由がないです。

LINE — 日本人の生活動線にあるのは魅力

9,500万人が使ってる。これはLINEの最大にして唯一の武器。

「いつも使ってるアプリからAIに話しかけられる」のは、確かに便利です。

ただし月200通の壁、サーバーが必要、セットアップの手間——と、AIエージェント用途ではハードル が多い。

LINEでAIエージェントを動かすなら、「AIからの通知は少なめ。自分から聞いたときだけ返事をくれれ ばいい」というライトな使い方がベストです。

一覧で比較するとこうなる

「でも日本でTelegram使ってる人いなくない？」

そう。日本でのTelegram利用率はわずか1%。

世界では10億人が使ってるのに、日本ではほぼ誰も使ってない。

理由はシンプルで、LINEが強すぎるから。

日本のスマホユーザーの9割以上がLINEを使ってて、「メッセージアプリ = LINE」になってる。

友達もLINEにいる。家族もLINEにいる。わざわざ別のアプリを入れる理由がない。

しかもTelegramは日本だと「匿名性が高い = ちょっと怪しい」ってイメージがある。

でも、ここで考えてほしいのは——

AIエージェントとの会話は、友達や家族との会話じゃない。

自分だけが使う「仕事道具」です。

仕事道具を選ぶとき、「友達が使ってるか」は関係ない。「自分にとって便利か」だけが判断基準。

テレグラムをインストールして、AIエージェント専用の「仕事部屋」として使う。 友達とのやりとり は今まで通りLINEで。

こう考えると、Telegramが日本でマイナーなことはデメリットにならない。

どう選ぶ？3パターンで整理

【画像: 個人・チーム・会社の3分岐フローチャート (04-choose-pattern.png)】

パターン1：個人でAIアシスタントが欲しい → Telegram一択。無料・簡単・制限なし。海外のセット アップガイドでも、まずTelegramから始める流れが主流

パターン2：チームやコミュニティでAIを共有したい → Discord。チャンネルで文脈を分けられるし、 無料。メンバー管理もしやすい

パターン3：会社のSlackにAIを入れたい → Slack。業務ツールの延長線上に置けるのは強い。ただし 有料プラン推奨

LINEは「通知を受け取るだけ」みたいなライトな用途ならアリ。 ChatworkとTeamsは、AIエージェン ト用途では現状おすすめしにくい。

まとめ：Telegramは「仕事道具として最適」だった

日本で流行ってない。周りに使ってる人がいない。なんか怪しい。

完全無料。サーバー不要。5分で動く。

「日本で流行ってるか」と「AIの仕事道具として優秀か」は、まったく別の話だったということで す。

友達とのチャットはLINE。 AIアシスタントとの仕事はTelegram。

この使い分けが、2026年の個人AI運用のスタンダードになるかもしれません。
