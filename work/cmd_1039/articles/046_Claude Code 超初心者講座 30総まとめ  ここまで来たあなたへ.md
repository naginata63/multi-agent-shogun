3つの編のロードマップ — 初級編#1-8（会話する力）→ 中級編#9-20（カスタマイズする力）→ 上級編#21-29（作る力）
このシリーズは、3つの編で構成されていました。
初級編（#1〜#8）
インストール、基本操作、プロンプトの書き方。
ここで身についたのは「Claude Codeと会話する力」です。 ターミナルを開いて、日本語で指示を出
して、結果を受け取る。 最初はおそるおそるだったのが、だんだん自然にできるようになったんじゃ
ないでしょうか。
中級編（#9〜#20）
CLAUDE.md、Hooks、MCP、Planモード、セッション管理。
ここで身についたのは「Claude Codeを自分好みにカスタマイズする力」。 毎回同じ説明をしなくてよ
くなったし、やってはいけない操作をガードできるようにもなりました。
道具を自分の手に馴染ませた、という感覚に近いかもしれません。
上級編（#21〜#29）
アプリ開発、デザイン、公開（デプロイ）、データ保存、トラブルシューティング。
ここで身についたのは「Claude Codeで実用的なものを作る力」です。 ToDoアプリ、家計簿アプリ。
動くものを作って、公開するところまでやりました。
振り返ってみると、けっこう遠くまで来てますよね。
初級編  → 会話できるようになった
中級編  → カスタマイズできるようになった
上級編  → 実用的なものを作れるようになった
