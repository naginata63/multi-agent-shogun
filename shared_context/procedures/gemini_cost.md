# Gemini 画像生成コスト管理

ガチャ無制限実行で **3,409枚生成 → 22,000円課金** (2026-03-28 cmd_28系) の教訓。

## 鉄則

1. **ガチャ上限**: 同一パネル構成の画像生成は最大 3 セット (殿許可なく追加禁止)
2. **1パネル試し打ち**: 全パネル生成前に P1 だけ生成 → 殿確認 → OK なら残り生成
   - NG項目 (ステーキ混入等) を全パネル生成前に発見する
3. **問題パネルだけ再生成**: 全パネル再生成禁止・問題パネルのみ `--skip-gen`+個別再生成
4. **ref_image Vision 分析キャッシュ**: `*_ref_vision.txt` 存在時は再分析しない (`--skip-vision`)
5. **バジェットアラート**: GCP で月額上限設定済・超過時は自動停止
