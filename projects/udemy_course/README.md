# Udemy講座作成プロジェクト

## 概要
Udemy講座の企画・制作・公開を行うプロジェクト。

## ディレクトリ構成
- `context/` — 企画資料、テーマ候補、ターゲット分析、競合調査
- `drafts/` — 台本・スクリプト・原稿のドラフト
- `slides/` — スライド（Marp/PowerPoint等）
- `videos/` — 収録動画・編集済み動画
- `materials/` — 受講者配布資料（サンプルコード、チートシート等）
- `scripts/` — 自動化スクリプト（字幕生成・編集自動化等）
