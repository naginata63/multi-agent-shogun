## 5. 競合対決表

### 5.1 対決表: Claude Code 入門領域

| 比較軸 | 本講座（初級編） | 「Claude Code 入門」（他講師・想定） | 「ChatGPT × VSCode 開発入門」（既存） |
|--------|----------------|------|------|
| 対象ツール | Claude Code CLI特化 | Claude Code概要〜入門 | ChatGPT + VSCode拡張 |
| 所要時間 | 5時間（完走しやすい） | 3-8時間（講座による） | 4-6時間 |
| 成果物 | 3点（CLAUDE.md+hook+MCP） | 概念理解のみ | プロンプト集 |
| hook/安全装置 | 1本の実hook作成 | なし | なし |
| MCP/記憶 | 接続体験あり | なし | なし |
| 中級編導線 | 明示（シリーズ化） | 単発 | 単発 |
| 価格 | 1,980円 | 1,200-2,400円 | 1,980-2,980円 |
| ペルソナ設計 | 4タイプ明示 | なし | なし |

### 5.2 対決表: AI開発入門領域

| 比較軸 | 本講座 | 「AI で開発を爆速化する」（初心者向けプロンプト中心） | 「Claude完全入門」（Udemy既存） |
|--------|--------|------|------|
| アプローチ | ツール使いこなし（CLI実践） | プロンプト技法中心 | Claude全体（Chat+Cowork+Code） |
| 実装深度 | hook作成+MCP接続 | プロンプト例示 | 各機能紹介 |
| Claude Code特化 | ✅ CLI専科 | ❌ ツール非特化 | △ Code部分のみ |
| 完走率目標 | 50%明示 | 不明 | 不明 |
| 環境要件 | 1人+1AI・特殊HWなし | ブラウザのみ | ブラウザ+VSCode |

### 5.3 差別性の要約

本講座の**3つの独自ポイント**:
1. **hook作成体験**: 他の入門講座にない「AIの安全装置を作る」ハンズオン
2. **シリーズ設計**: 初級→中級の明示的導線でLTV最大化
3. **3点成果物**: 概念理解で終わらず、持ち帰れる成果物がある
